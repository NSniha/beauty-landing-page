# Velvety — Beauty & Skincare Landing Page

A modern, fully responsive beauty brand landing page built with **React** and **Tailwind CSS**. Designed for a fictional organic skincare company called **Velvety**, featuring product showcases, filterable product grids, testimonials, and more.

> 🌿 Live Demo: [https://radiant-biscuit-b7350a.netlify.app/](#)

## Sections

| Section | Description |
|---|---|
| **Hero** | Full-screen headline with CTA and product visual |
| **Features** | 100% Organic · Fits your skin · Easy to use |
| **Featured Products** | Product cards with name, price, and star ratings |
| **About** | Brand story and philosophy |
| **As Seen In** | Press logos — Vogue, Forbes, Women's Health |
| **All Products** | Filterable product grid with categories |
| **Skin Diagnosis** | CTA section for personalized skincare |
| **Testimonials** | Customer review with star rating |
| **Promotions** | Loyalty program, gifting, and wellness cards |
| **Newsletter** | Email subscription with discount offer |
| **Footer** | Multi-column footer with shop, help & store links |

---

## Tech Stack

- **React 18** — component-based UI
- **Tailwind CSS** — utility-first styling
- **Vite** — fast dev server and build tool

---

## Getting Started

```bash
# Clone the repo
git clone https://github.com/NSniha/beauty-landing-page.git

# Navigate into the project
cd beauty-landing-page

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
